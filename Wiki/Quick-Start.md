# Quick Start

This page gets you from install to productive dictation quickly.

## Requirements

- macOS 13.3 or newer
- Microphone permission
- Accessibility permission
- Speech Recognition permission (only if using Apple Speech)

## Install

1. Download the latest `.dmg` or app build from [Releases](https://github.com/manikv12/OpenAssist/releases).
2. Open `Open Assist.app`.
3. Confirm the menu-bar icon appears.

## First-run checklist

Open `Settings` from the menu bar and confirm:

1. `Accessibility` is granted.
2. `Microphone` is granted.
3. `Speech Recognition` is granted (Apple Speech engine only).
4. Your preferred shortcuts are set.

## First dictation

1. Click into any text field.
2. Hold `Option + Command + Space`.
3. Speak naturally.
4. Release to insert text.

## Pick your speech engine

In `Settings -> Speech and Input -> Speech Engine` choose:

- Apple Speech: ready immediately with system APIs.
- Whisper.cpp: fully local once the model is downloaded.

## Optional: turn on AI rewrite

In `Settings -> AI and Models`:

1. Enable AI rewrite.
2. Choose provider (local Ollama or cloud provider).
3. Set rewrite strength: `Light`, `Balanced`, or `Strong`.
