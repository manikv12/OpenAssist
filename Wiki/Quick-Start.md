# Quick Start

This page gets you from install to your first useful task as fast as possible.

## What you need

- macOS 13.3 or newer
- `Microphone` permission for voice input
- `Accessibility` permission for shortcuts and text insertion
- `Speech Recognition` if you use Apple Speech
- `Automation / Apple Events` if you want browser or app control

## Install

1. Download the latest build from [Releases](https://github.com/manikv12/OpenAssist/releases).
2. Open `Open Assist.app`.
3. Approve the permissions you need.
4. Confirm the menu bar icon appears.

## First setup

1. Open `Settings`.
2. Check `Accessibility` and `Microphone`.
3. Check `Speech Recognition` if you use Apple Speech.
4. Open `AI Studio` to connect a provider or set up local AI.
5. Open `Automation` only if you want browser or app actions.

## First assistant task

1. Open the menu bar popover.
2. Choose `Open Assistant` or `Speak Assistant Task`.
3. Try a simple request.

Examples:

- "Summarize these notes."
- "Rewrite this message to sound more friendly."
- "Open Bluetooth settings."

## First dictation

1. Click into any text field.
2. Hold `Option + Command + Space`.
3. Speak naturally.
4. Release to insert the text.

## Default shortcuts

- Hold-to-talk: `Option + Command + Space`
- Continuous dictation: `Control + Option + Command + Space`
- Paste last transcript: `Option + Command + V`
