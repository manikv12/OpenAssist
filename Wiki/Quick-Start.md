# Quick Start

This page gets you from install to your first useful assistant task.

## Requirements

- macOS 13.3 or newer
- Microphone permission
- Accessibility permission
- Speech Recognition permission if you use Apple Speech
- Automation / Apple Events permission if you want browser or app control

## Install

1. Download the latest release from [Releases](https://github.com/manikv12/OpenAssist/releases).
2. Open `Open Assist.app`.
3. Approve the permissions you are asked for.
4. Confirm that the menu bar icon appears.

## First-run checklist

Open `Settings` from the menu bar and confirm:

1. `Accessibility` is granted.
2. `Microphone` is granted.
3. `Speech Recognition` is granted if you use Apple Speech.
4. `Automation / Apple Events` is granted if you want browser or app control.
5. Your shortcut keys are set the way you want.
6. Open `AI Studio` if you want local or cloud AI setup.

## First assistant task

1. Open the menu bar popover.
2. Choose `Open Assistant` for typed work or `Speak Assistant Task` for voice-first work.
3. Type or say a simple request.
4. Review the answer before using it elsewhere.

Example prompts:

- "Summarize these notes."
- "Rewrite this message to sound more friendly."
- "Open my project board in Chrome."

## First dictation

1. Click into any text field.
2. Hold `Option + Command + Space`.
3. Speak naturally.
4. Release to insert the text.

## Helpful shortcuts

- Hold-to-talk: `Option + Command + Space`
- Continuous dictation: `Control + Option + Command + Space`
- Paste last transcript: `Option + Command + V`

## Optional: turn on AI rewrite

1. Open `Settings -> AI & Models`.
2. Turn on AI rewrite.
3. Choose a provider.
4. Pick a rewrite strength such as `Light`, `Balanced`, or `Strong`.
5. If you want local-only AI, use `AI Studio` to install or repair local models.

## Optional: browser or app control

1. Open `Settings -> Automation`.
2. Allow `Automation / Apple Events`.
3. Pick a browser profile if you want browser control.
4. Use `Agentic` mode when you want the assistant to take actions for you.
